# Security Policy

## Supported Versions

This is a graduation project. Only the latest version on `main` is maintained.

| Version | Supported |
|---------|-----------|
| latest (main) | ✅ |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Instead, report it privately by emailing the repository owner via the contact on their GitHub profile, or by using GitHub's private vulnerability reporting:

1. Go to the **Security** tab of this repo
2. Click **Report a vulnerability**
3. Fill in the details

You can expect a response within 7 days. If the vulnerability is confirmed, a fix will be prioritized and credited to you in the release notes.

## Scope

This project is designed for **local deployment** in an educational environment. It is not intended for production use without additional hardening. Known limitations are documented in [SECURITY_REVIEW.md](SECURITY_REVIEW.md).
